CommandDescription
mvn cleanDeletes all files in target directory or directories (for multi-module projects)
mvn compileCompiles Project
mvn war:warBuild war file
mvn testRuns all tests in src/test/java. Use "-Dtest=ClassName" (not fully-qualified) to run individual tests.
mvn surefire-report:reportGenerate html report for tests
mvn surefire:testRun surefire tests
mvn jetty:run-warPackages and deploys your application to Jetty, reachable at http://localhost:8080
mvn cargo:start -Dcargo.wait=trueapplication will be deployed to Tomcat and available at http://localhost:8081/applicationname-version.
You can also add the Maven 2 Tomcat Plugin to your pom.xml and use mvn tomcat:run and mvn tomcat:run-war
mvn packageCreates a WAR or JAR depending on your project type
mvn siteCreates project site and reports
mvn integration-testRuns UI tests in Tomcat using Cargo
mvn --helpSee full list of optional commands
mvn dbunit:export -Ddest=some-data-sample-data.xmlTo generate dbunit data from database existing entries
Command SwitchesDescription
mvn ... -UChecks for updated plugins and downloads if they exist
mvn ... -oWork offline
mvn ... -DskipTests=trueDo not run tests
mvn ... -Dcargo.wait=truePackages and deploys your application to active Cargo profile (Tomcat 5.5.x by default), reachable at http://localhost:8080/yourapp-version
mvn ... -Dsurefire.useFile=falseUse if you want to see test failures in your console (like Ant)
mvn ... -Dmaven.surefire.debugif you want to open a debugger on port 5005 w/ suspend=y
