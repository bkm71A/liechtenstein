3.       Add  logarithmic scale graph:

   To make the logarithmic or dB scale on Y axis we need to recalculate the data to dB (decibel) units. The formula is as follows:

A[dB] = 20log(A/A[ref]),

where A is our acceleration data, and A[ref] is a reference level that must be settable in the user preferences - "Reference level for dB scale" (separate for accel. and velocity). The default is 1 micro g. In other words, use A[ref] = 10^-6 g for the default.

For velocity

V[dB] = 20log(V/V[ref]),

and the default for V[ref] is 10^-9 m/s or 3.937*10^-8 in/s.

On the graph the scale will remain linear, only the units will switch to dB and the data values to be plotted will be either A[dB] or V[dB].
